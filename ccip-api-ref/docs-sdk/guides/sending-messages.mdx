# Sending Messages

## Message Structure

```typescript
interface CCIPMessageInput {
  receiver: string          // Destination address (encoded for target chain)
  data: string              // Arbitrary data payload (hex string)
  tokenAmounts: {           // Tokens to transfer
    token: string           // Source token address
    amount: bigint          // Amount in smallest unit
  }[]
  feeToken: string          // Fee payment token (address or zero for native)
  extraArgs: string         // Encoded extra arguments (gas limit, etc.)
}
```

## Simple Message

Send arbitrary data to a contract on another chain:

```typescript
import { EVMChain, encodeExtraArgs, networkInfo } from '@chainlink/ccip-sdk'
import { Wallet } from 'ethers'
import { toHex } from 'viem'

const wallet = new Wallet(process.env.PRIVATE_KEY!)
const source = await EVMChain.fromUrl('https://rpc.sepolia.org')

const message = {
  receiver: '0xReceiverContract...',
  data: toHex('Hello from Sepolia!'),
  tokenAmounts: [],
  feeToken: '0x0000000000000000000000000000000000000000', // Native ETH
  extraArgs: encodeExtraArgs({
    _tag: 'EVMExtraArgsV2',
    gasLimit: 200000n,
    allowOutOfOrderExecution: false,
  }),
}

const router = source.network.router
const destSelector = networkInfo('avalanche-testnet-fuji').chainSelector

const fee = await source.getFee(router, destSelector, message)
console.log('Fee:', fee, 'wei')

// Include fee in message and provide wallet for signing
const request = await source.sendMessage(
  router,
  destSelector,
  { ...message, fee },
  { wallet }
)
console.log('Sent in tx:', request.tx.hash)
console.log('Message ID:', request.message.messageId)
```

## Token Transfer

Transfer tokens cross-chain:

```typescript
import { EVMChain, encodeExtraArgs, networkInfo } from '@chainlink/ccip-sdk'
import { Wallet } from 'ethers'

const wallet = new Wallet(process.env.PRIVATE_KEY!)
const source = await EVMChain.fromUrl('https://rpc.sepolia.org')

const LINK_TOKEN = '0x779877A7B0D9E8603169DdbD7836e478b4624789' // LINK on Sepolia

const message = {
  receiver: '0xRecipientAddress...',
  data: '0x', // No data, just token transfer
  tokenAmounts: [
    {
      token: LINK_TOKEN,
      amount: 1000000000000000000n, // 1 LINK (18 decimals)
    },
  ],
  feeToken: LINK_TOKEN, // Pay fee in LINK
  extraArgs: encodeExtraArgs({
    _tag: 'EVMExtraArgsV2',
    gasLimit: 0n, // No receiver execution needed
    allowOutOfOrderExecution: true,
  }),
}

const destSelector = networkInfo('avalanche-testnet-fuji').chainSelector
const router = source.network.router

const fee = await source.getFee(router, destSelector, message)
console.log('Fee:', fee, 'LINK wei')

// SDK handles token approvals automatically via approveMax option
const request = await source.sendMessage(
  router,
  destSelector,
  { ...message, fee },
  { wallet, approveMax: true }
)
console.log('Sent in tx:', request.tx.hash)
```

Before sending tokens, ensure you have approved the Router contract to spend your tokens, or use `approveMax: true` to auto-approve.

## Fee Estimation

Estimate fees before sending:

```typescript
import { EVMChain, networkInfo } from '@chainlink/ccip-sdk'

const source = await EVMChain.fromUrl('https://rpc.sepolia.org')

// Fee in native token
const nativeMessage = { ...message, feeToken: '0x' + '0'.repeat(40) }
const nativeFee = await source.getFee(router, destSelector, nativeMessage)

// Fee in LINK
const linkMessage = { ...message, feeToken: LINK_TOKEN }
const linkFee = await source.getFee(router, destSelector, linkMessage)

console.log('Native fee:', nativeFee, 'wei')
console.log('LINK fee:', linkFee, 'wei')
```

Fees depend on destination chain gas costs, token transfer complexity, message data size, and current gas prices.

## Unsigned Transactions

Generate unsigned transactions for offline signing or multi-sig wallets:

```typescript
import { EVMChain, networkInfo } from '@chainlink/ccip-sdk'

const source = await EVMChain.fromUrl('https://rpc.sepolia.org')
const senderAddress = '0xYourWalletAddress...'

const router = source.network.router
const destSelector = networkInfo('avalanche-testnet-fuji').chainSelector

// Get fee first
const fee = await source.getFee(router, destSelector, message)

// Generate unsigned transaction (sender address is required)
const unsignedTx = await source.generateUnsignedSendMessage(
  senderAddress,  // sender address

  destSelector,
```

## Complete Example

Send data and tokens with fee buffer:

```typescript
import {
  EVMChain,
  encodeExtraArgs,
  networkInfo
} from '@chainlink/ccip-sdk'
import { Wallet } from 'ethers'
import { toHex, parseEther } from 'viem'

async function sendCrossChainMessage() {
  const wallet = new Wallet(process.env.PRIVATE_KEY!)
  const source = await EVMChain.fromUrl('https://rpc.sepolia.org')

  const router = source.network.router
  const destSelector = networkInfo('avalanche-testnet-fuji').chainSelector

  const message = {
    receiver: '0xReceiverContract...',
    data: toHex(JSON.stringify({ action: 'deposit', user: '0x...' })),
    tokenAmounts: [
      {
        token: '0x779877A7B0D9E8603169DdbD7836e478b4624789', // LINK
        amount: parseEther('0.1'),
      },
    ],
    feeToken: '0x0000000000000000000000000000000000000000', // Native ETH
    extraArgs: encodeExtraArgs({
      _tag: 'EVMExtraArgsV2',
      gasLimit: 300000n,
      allowOutOfOrderExecution: false,
    }),
  }

  const fee = await source.getFee(router, destSelector, message)
  console.log('Estimated fee:', fee, 'wei')

  // Add 10% buffer for gas price fluctuations
  const feeWithBuffer = (fee * 110n) / 100n

  // sendMessage returns CCIPRequest containing tx and message info
  const request = await source.sendMessage(
    router,
    destSelector,
    { ...message, fee: feeWithBuffer },
    { wallet, approveMax: true }
  )

  console.log('Transaction hash:', request.tx.hash)
  console.log('Message ID:', request.message.messageId)

  return request
}
```
